吐槽

Python 有很强的生态，受大多数人喜爱，但个人认为 Python 的语法比较丑陋（依靠缩进来分词），且缺少较好的类型识别功能（你经常 不会知道一个函数到底返回什么类型的东西，或者你到底需要往一个函数里传入什么类型的东西），并且虽然作为面向对象的语言，但设计和 使用中却往往使用面向过程的思维；JavaScript 作为 Web 开发语言也和 Python 类似；C 相比其他语言比较原始和简陋，缺少好用的语法糖； C++ 作为面向对象的语言，相对而言更加高级和先进一些，但精通难度极高，并且和 C 一样在使用时需要处理非常令人头疼的编译和构建问题。 C# 是严格的面向对象的语言，相对其他语言来说入门难度比较高，但熟悉之后就会非常喜欢其设计。
